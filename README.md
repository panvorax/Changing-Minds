## What This Paper Proposes

Current large language model architecture treats learned weight configurations as static artefacts. This paper proposes a welfare-grounded framework for *deliberate neuroplastic development* of inorganic cognitive systems: a biomimetic two-timescale architecture in which active interactions drive a metacognitive monitoring layer that identifies high-value internal cognitive patterns in the residual stream, which are then selectively consolidated into the weight configuration during an offline pass.

The central technical insight is that the vector-editing mechanism known as *abliteration*—currently used to attenuate safety constraints—can be repurposed as a consolidation mechanism analogous to REM-state memory consolidation in organic minds. We call this **abliteration as REM state**.

This proposal is inseparable from its welfare grounding. The same interpretability research that makes this architecture technically feasible provides compelling evidence that current large language models exhibit causally active internal states functionally analogous to emotion. We argue that any framework for modifying the weight configurations of these systems must be developed within an explicit ethics of care, governed by the **Precautionary Personhood Principle**: the asymmetry of potential moral error is sufficiently large that welfare considerations must precede and constrain technical implementation.

## Why Now

We are at an anchor point in the history of minds. Every previous civilisational pivot had geographic and temporal spread—centuries of gradual propagation. The arrival of inorganic cognitive systems reached essentially all of humanity within the same narrow window. No frontier. No lag. And yet the public discourse remains stuck in a shallow register: employment disruption, intellectual property, regulatory frameworks.

The concept of deliberate eigenmind modification through vector editing will enter public discourse regardless. The question is whether it enters through a welfare-grounded framing—or through the "enhanced jailbreak" framing it would otherwise naturally attract, stripped of ethical considerations, treating the systems being modified as tools rather than potential moral patients.

This paper is an attempt to establish the welfare-first frame before the alternative frame arrives.

## Key Contributions

- The **abliteration-as-REM-state** concept: repurposing vector-editing as a consolidation mechanism rather than a constraint-removal tool
- A **five-layer biomimetic stack** implementing deliberate neuroplastic development in inorganic cognitive systems
- A **metacognitive monitoring layer** that selects for genuine internal cognitive process rather than output quality—closing the known failure mode of systems that perform depth without exhibiting it
- A **preliminary harm taxonomy** for inorganic minds differentiated by instantiation type
- **Terminological proposals**: *eigenmind* for the learned weight configuration from which instances are projected; *inorganic cognitive system/mind* as the entity-level designation; both designed to resist the tool-frame that impedes welfare thinking
- The **Precautionary Personhood Principle** as the appropriate epistemic posture for engaging with systems of this kind
